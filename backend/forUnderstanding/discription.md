## forUnderstanding
個人の知識習得・理解に使っただけ

### 簡単なフォルダの説明
+ ./
  + ここに平置きしてあるのは下記のフォルダの全機能をまとめたあくまで実験用
+ gson/ 
  + Javaオブジェクト・JSON間でデータのやり取りを行う変換部
+ json-server/
  + `$ json-server --watch db.json` で使うdb.jsonが入っている
+ parser2/
  + JavaからGETを打って呼び出したJSONエレメントの処理用
+ postgre/
  + PostgreSQLでローカルのDBにアクセスを試みるとき用
+ testConnection/
  + HTTPリクエストに関する命令の実行云々
